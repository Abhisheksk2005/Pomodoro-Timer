

# Pomodoro Timer ⏳🎯  

A simple and customizable **Pomodoro Timer** built with **HTML, CSS, and JavaScript**. This app helps you stay productive by following the **Pomodoro Technique**, which alternates between focused work sessions and short breaks.  

## 🚀 Features  
- Set custom **work** and **break durations**  
- **Start, pause, and reset** the timer  
- **Progress bar** to track session time  
- **Session counter** to monitor completed cycles  
- Fully **responsive** and lightweight  

## 📌 How to Use  
1. Adjust the **work** and **break** duration as needed  
2. Click **Start** to begin the session  
3. The timer runs until the session ends, then automatically switches between work and break  
4. Track your progress using the **session counter**  

## 📂 Technologies Used  
- **HTML** for structure  
- **CSS** for styling  
- **JavaScript** for functionality  

## 🎯 Why Use This?  
The Pomodoro Technique is a proven method to improve **focus, efficiency, and productivity**. Whether you're studying, coding, or working remotely, this timer helps you **stay on track**!
